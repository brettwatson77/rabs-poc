## 7. Data-Flow Debugging Checklist  

1. **Connection Type**  
   * On backend start you must see `Using PostgreSQL database: rabspocdbâ€¦`  
   * If you see `SQLite` -> `.env` mis-set or wrapper not imported.

2. **Missing Table**  
   ```
   error: relation "pending_enrollment_changes" does not exist
   ```
   âœ run `database/migrations/` SQL or manual `CREATE TABLE` (see Â§4).

3. **Invalid UUID**  
   ```
   invalid input syntax for type uuid: "with-schads"
   ```
   âœ Frontend hard-coded ID â€“ inspect call stack, fix API param.

4. **Empty Schedule**  
   * Check `master_schedule_items`, `loom_instances`, `event_card_map`.  
   * Run `projectorService.project()` manually:  
     ```bash
     node scripts/prepare-launch.js --project
     ```

5. **Wrapper Edge-Case**  
   * If service uses raw `client.query` youâ€™ll get PG `$1` placeholders; convert `?` or bypass wrapper.

6. **CORS / Port Mismatch**  
   * Backend 3009, Frontend 3008 â€“ update `frontend/src/api/api.js` baseURL.
